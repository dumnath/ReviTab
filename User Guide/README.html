Introduction

RéviTab est une application fonctionnant sur Windows permettant d'éditer des fichiers au format CSV et de s'entraîner dessus.

Guide d'utilisation

1. Installation de l'application

RéviTab est une application Windows compilée au format .exe à l'aide de PyInstaller en onefile : c'est-à-dire que l'application fonctionne uniquement sur Windows et ne contient qu'un seul fichier. Elle ne nécessite aucune autre installation. L'application peut être stockée à n'importe quel endroit sur l'ordinateur.

Lors de la première utilisation, un dossier .revitab sera créé. Celui ci contient le fichier de configuration de l'application appelé config.toml. Il contient également les fichiers de langue (par défaut : english.toml et francais.toml). Vous pouvez rajouter vos propres fichiers de langue, tout en faisant attention aux noms des clés des éléments du fichier (CONSEIL : Copier un des fichiers par défaut et changer le texte). Si le dossier est supprimé, déplacé ou renommé, il sera automatiquement récréé avec les paramètres par défaut de l'application.

2. Fonctionnalités de l'application

2.1 Interface générale

RéviTab fonctionne selon un système d'onglets. Vous pouvez donc ouvrir plusieurs fichiers en même temps, ainsi que créer plusieurs exercices. En fonction du type d'onglet ouvert (Accueil, Documentation, Exercice, Edition), la barre d'outils change. Vous n'avez donc pas accès aux mêmes fonctionnalités en fonction de l'onglet ouvert.

2.2 Edition de fichier CSV

Pour ouvrir un fichier CSV, allez dans le menu Fichier, et sélectionnez le menu Ouvrir... (raccourci : Ctrl+O). Une fenêtre s'ouvrira, vous permettant de choisir un fichier à ouvrir. Vous ne pouvez ouvrir que des fichiers CSV.

Vous pouvez également créer un nouveau fichier, en sélectionnant l'option Nouveau du menu Fichier (raccourci : Ctrl+N).

Le fichier CSV choisi est affiché sous forme d'un tableau. Vous pouvez ajouter/retirer des lignes et des colonnes, ainsi que changer les en-têtes des colonnes.

Pour sauvegarder le fichier, appuyer sur l'icône de sauvegarde de la boîte d'outils ou allez dans Fichier, Sauvegarder (raccourci Ctrl+S). S'il s'agit d'un nouveau fichier, une fenêtre demandant de choisir la localisation du CSV apparaîtra. Si le fichier a été modifié et non sauvegardé, une confirmation sera demandée avant la fermeture de l'onglet ou de l'application.

Pour insérer un ß, utiliser le raccourci Ctrl+Shift+S.

Vous pouvez également créer des exercices à partir des fichiers ouverts (voir section suivante).

2.3 Création d'exercices

RéviTab permet de générer des exercices à partir de fichiers CSV. Pour pouvoir créer un exercice, vous devez d'abord ouvrir le fichier sur lequel vous souhaitez vous entraîner, puis, vous appuyez sur le bouton Entraînement (raccourci Ctrl+T). Pour pouvoir correctement créer l'exercice, assurez vous que le fichier CSV est bien enregistré.

Lorsque vous créez l'exercice, vous pouvez choisir le nombre de questions de l'exercice ainsi que les lignes du CSV qui seront utilisées lors de sa création.

L'exercice généré prend la forme d'un tableau, dont seule une case par ligne est donnée. Le but de l'exercice est donc de retrouver les autres éléments de la ligne. Vous pouvez choisir la colonne montrée (la première, la dernière ou une aléatoire).

2.4 Mode exercice

Le but d'un exercice est de remplir toutes les cases vides du tableau, en s'aidant de la case montrée.

Lorsque vous faîtes un exercice, la boîte à outils propose deux options : Vérifier et Recharger.

  • La première option permet de vérifier les réponses rentrées dans le tableau. Le score est alors calculé de la manière suivante : 1pt pour chaque cellule correcte, sur le total de réponses attendues.
  • La deuxième option permet de recréer un nouvel exercice à partir du même fichier (raccourci Ctrl+R). Quand vous rechargez un exercice, vous pouvez changer les paramètres de l'exercice (nombre de questions et lignes utilisées).

Lors de la vérification, l'application compare les réponses avec une tolérance à la casse et aux espaces par défaut (cette option peut être modifiée dans les paramètres, voir section suivante). Les bonnes réponses sont surlignées en vert, les mauvaises en rouge. Un GIF peut apparaître en fonction du score obtenu (cette option est également optionnelle).

2.5 Paramétrer l'application

Pour pouvoir changer les paramètres de l'application, allez dans le menu Paramètres et sélectionnez l'option Paramètres... (raccourci Ctrl+,). Cela ouvrira une fenêtre permettant de changer les paramètres. Les paramètres modifiables sont les suivants :

  • La langue de l'application
  • Le style de l'application
  • Lors de la vérification des réponses :
    • La tolérance à la casse
    • La tolérance aux espaces
  • L'apparition ou non d'un gif lors de l'affichage du score
  • La colonne montrée lors de l'exercice
  • Le séparateur utilisé lors de la lecture et de l'écriture des fichiers CSV

Tous ces paramètres peuvent être réinitialisés avec l'option Réinitialiser les paramètres, du menu Paramètres.

Tous les paramètres sont enregistrés dans le fichier config.toml, dans le dossier .revitab.

2.6 Licences et Guide utilisateur

Pour pouvoir accéder aux licences de l'application et des icones utilisés, allez dans le menu Aide, A propos. Pour ouvrir le fichier README, sélectionnez, dans le même menu, Guide Utilisateur (raccourci F1). Les informations sont donc intégrées directement dans l'application.
